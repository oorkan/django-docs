Installation

Python

First, we need to check if we already have the python installed. python3 --version || python --version

Django 3 requires python >= 3. If our version is smaller then we need to upgrade it. sudo apt install python3 It's also possible that our system will contain two versions of python, e.g. >=2 and >= 3. That's ok.

 

Installing Django's official release

For this purpose, we'll need to have a package called pip installed on our system. Pip is a Python's package manager system and also recursive abbreviation meaning Pip Installs Packages. sudo apt install python3-pip After installing pip, we can now continue to installation of Django. python3 -m pip install Django Where the argument m stands for mod and is running library module as a script.

To verify that the Django is successfully installed we can run the following command: python3 -m django --version
